This is an image (based on Alpine Docker Image) with Neovim installed, including Python3 support 
 and a bunch of plugins. 

The list of currently used plugins :

```
Plug 'chriskempson/base16-vim'
Plug 'ctrlpvim/ctrlp.vim' " fuzzy file finder
Plug 'vim-airline/vim-airline' " fancy statusline
Plug 'vim-airline/vim-airline-themes' " themes for vim-airline
Plug 'tomtom/tcomment_vim' " comment out things
Plug 'tpope/vim-surround' " mappings to easily delete, change and add such surroundings in pairs, such as quotes, parens, etc...
Plug 'junegunn/goyo.vim', { 'on': 'Goyo' } " distraction-free writing
Plug 'junegunn/limelight.vim', { 'on': 'Limelight' } " focus tool. Good for presenting with vim
" Plug 'scrooloose/nerdtree'

Plug 'Shougo/deoplete.nvim'
```
